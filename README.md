# 🎓 Student Management System

A full-featured School Management Web Application built with **Spring Boot**, **Thymeleaf**, **MySQL**, and **Spring Security**.

---

## 🔗 Tech Stack

| Technology | Description |
|---|---|
| Java 21 | Programming Language |
| Spring Boot 3.5 | Backend Framework |
| Spring Security | Authentication and Authorization |
| Spring Data JPA | Database ORM |
| MySQL | Database |
| Thymeleaf | Template Engine |
| Bootstrap 4 | Frontend Styling |
| Chart.js | Dashboard Charts |

---

## ✅ Features

### 🔐 Security
- Login and Signup with BCrypt password hashing
- Role based access control — Admin and Viewer
- Forgot password with security question and answer
- Edit own profile — username and password

### 👨‍🎓 Student Management
- Add, Update, Delete, View students
- Student photo upload
- Gender field
- Search students by keyword
- Pagination — 5 students per page
- Student detail view page
- Student ID card generator with print option
- Export student list to CSV
- Print student list

### 📚 Course Management
- Add and delete courses with course code
- Assign multiple courses to students

### ✅ Attendance
- Mark daily attendance for all students
- View attendance records per student
- Present and Absent count
- Attendance percentage calculation

### 📊 Grades
- Add marks per student per course
- Auto calculate grade — A+, A, B, C, D, F
- View average marks per student

### 💰 Fee Management
- Add fee records per term
- Track amount due, amount paid, balance
- Auto status — PAID, PARTIAL, UNPAID
- Fee overview for all students

### 📅 Timetable
- Weekly timetable schedule grid
- Add slots by day, course, start time, end time, room number
- Delete slots

### 📢 Announcements
- Admin posts notices and announcements
- All users can view announcements
- Posted by and posted date shown

### 📊 Dashboard
- Total students count card
- Total courses count card
- Students with and without photo count
- Gender breakdown pie chart
- Students per course bar chart
