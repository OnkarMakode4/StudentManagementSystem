# Student Management System

Built with Spring Boot, Thymeleaf, MySQL, Spring Security

## Features
- Login and Signup with BCrypt
- Role based access (Admin and Viewer)
- Student CRUD with photo upload
- Course management
- Attendance tracking
- Grades management
- Fee management
- Timetable
- Announcements
- Dashboard with charts
- Export CSV
- Student ID card
- Pagination and search
- Forgot password
- Profile management
